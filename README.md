# 使用方式

目前仅支持Windows系统

### 运行方法一：

`pip install -r requirements.txt`   安装必备的模块

`python .\Cubit_main.py`   启动GUI界面

> 如果无法启动可以尝试启动方法二

### 运行方法二：

在**Pycharm**界面，直接运行**main**文件

缺少的模块需自行安装
